# Scratch Paper
「自分の中でもやもやしている気持ち・悩み」を気軽に吐き出せるツイートアプリ。学習の一環で作成しました。
Webアプリの構造を学ぶため、フレームワークを使用せずサーブレットで作成しました。
レスポンシブ対応しているため、スマートフォンからも利用できます。
